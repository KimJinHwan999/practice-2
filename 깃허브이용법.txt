1. 명령 프롬프트에서 git config --global 로 컴퓨터 전체에서 동일 옵션 사용하도록 설정

2. git config --global user.name qwerty0310 / git config --global user.email wlsghks5438@gmail.com

3. 원하는 폴더를 만들고, 깃 저장소를 다운로드 받아서 동기화 시키기

4. 깃 저장소의 주소를 저장

5. 프롬프트에서 원하는 폴더로 이동

6. git clone https://github.com/KimJinHwan999/Hello-github.git (소스코드 그대로 가져오기)

7. 리포지터리 폴더로 들어가기 (내부에 .git 폴더 만들어진 것 확인)

8. 원하는 파일 생성하기

9. git add (원하는 파일명) (파일 추가)

10. git commit -m "Add Text File [document.txt]" -> 원하는 커밋명으로 설정 (컴퓨터에만 적용되고 허브에는 적용 안됨)

11. git push -> 깃허브에 푸시해서 저장시키기
